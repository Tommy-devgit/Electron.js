# Electron.js

Desktop application boilerplate using Electron + React + TypeScript + Vite.

## Overview

This repository combines:

- Electron main process (`src/electron`)
- React renderer UI (`src/ui`)
- IPC bridge through a preload script (`src/electron/preload.cts`)

The app currently polls system metrics (CPU, RAM, storage) in the Electron process and sends updates to the renderer.

## Tech Stack

- Electron
- React
- TypeScript
- Vite
- electron-builder

## Requirements

- Node.js 20+
- npm

## Getting Started

```bash
npm install
npm run dev
```

`npm run dev` starts:

- Vite dev server on `http://localhost:5123`
- Electron app pointing to that dev server

## Scripts

- `npm run dev`: run renderer + Electron in development
- `npm run dev:react`: run Vite only
- `npm run dev:electron`: transpile Electron code and start Electron in dev mode
- `npm run transpile:electron`: compile `src/electron` to `dist-electron`
- `npm run build`: type-check/build TypeScript and build React app to `dist-react`
- `npm run lint`: run ESLint
- `npm run preview`: preview Vite production build
- `npm run dist:win`: build Windows package (portable + msi)
- `npm run dist:mac`: build macOS package (arm64 dmg)
- `npm run dist:linux`: build Linux package (x64 AppImage)

## Project Structure

```text
src/
  electron/
    main.ts            # Electron main process + BrowserWindow + IPC handlers
    preload.cts        # Safe bridge (contextBridge) between renderer and Electron
    resourceManager.ts # System metrics polling + static system data
    pathResolver.ts    # Preload path resolution for dev/prod
    utils.ts           # Environment helpers
  ui/
    App.tsx            # React app entry UI
    main.tsx           # React bootstrap
```

## Build Output

- `dist-react/`: Vite renderer build
- `dist-electron/`: transpiled Electron files
- `dist/`: installer artifacts generated by electron-builder

## IPC Surface

Exposed on `window.electron` by preload:

- `subscribeStatics(callback)`: subscribes to live `statistics` updates
- `getStaticData()`: fetches static machine info from main process

## Packaging

Build configuration is in `electron-builder.json`.

Configured targets:

- Windows: `portable`, `msi`
- macOS: `dmg` (`arm64`)
- Linux: `AppImage` (`x64`)
